# DaggyD

A minimal, lightweight Python package.

## Installation

```sh
pip install -e .
```

## Usage

```python
import DaggyD
print(DaggyD.example_function())  # Expected output: "Hello from DaggyD!"
```

## Running Tests

```sh
python -m unittest discover DaggyD/tests
```

Expected output:

```
.
----------------------------------------------------------------------
Ran 1 test in 0.001s

OK
```
